## Stack

- Astro 7
- Tailwind CSS 4
- IBM Plex Sans + JetBrains Mono

## Desarrollo local

Requisitos:

- Node.js 22.12 o superior
- pnpm

```bash
pnpm install
pnpm dev
```

La aplicación queda disponible en `http://localhost:4321`.
